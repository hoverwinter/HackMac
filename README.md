# HackMac

## 2021/10/16
- 系统：macOS Big Sur 11.6
- 引导：OpenCore 0.7.4
- 工具
  - 16GB U盘，Mac 制作安装盘
  - MountEFI
  - GenSMBIOS
  - ProperTree

### 硬件配置
#### 影响安装
- CPU：Intel Core i5 10400
- 主板：MSI B460M Mortar WIFI
- 显卡：UHD630
- 网卡：FV-T919 （板载Intel AX200未驱动）
- M.2：Intel 512GB （Ubuntu） & 镁光 1TB （MacOS）

#### 其他配置
- 机箱：乔思伯 V9
- 散热：乔思伯 CR1400（塔式风冷）
- 风扇：乔思伯 FR925
- 内存：ADATA 8GB * 2

### 备注
- Intel 网卡未驱动
- 无独立显卡
- 一键黑苹果：MSI D.T.M. 模式（手动设置相关 BIOS 也可以）
- Boot Debug 日志未关闭
